## Featured Projects

### Document Intelligence Studio V2

Production-grade document intelligence platform for extracting, validating, and structuring information from scanned forms and multi-page documents using OCR, Computer Vision, PostgreSQL, Redis, Celery, Docker, Prometheus, and Grafana.

### EmoBhaava

Multilingual NLP system for sarcasm and sentiment analysis in Kannada, Romanized Kannada, and Kannada-English code-mixed text using transformer architectures.

### Cloud-Based NAAC Data Management System

Enterprise accreditation management platform featuring secure document storage, workflow automation, search capabilities, and role-based access control.
